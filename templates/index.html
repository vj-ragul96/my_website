Projects

Linear Actuator Monitoring of Vehicle Assembly line using ML Approaches:






This project was based on the development of a condition-based maintenance system for a rack and pinion drive in the marriage station of a Jaguar-Land Rover manufacturing plant. The key skills used during the project were FMECA, Digital twin using Simulink, Testing of the rig (which works similar to the marriage station), collection of information/data from different sensors, signal translation using MATLAB, Detection Algorithm development and Cost Analysis.

  • The rig was tested under 116 conditions and 537 data sets which includes Current, Accelerometer, UltraSonic and Vibration Sensors were collected and analysed.
  • Different kinds of features like RMS, Kurtosis, Mean, Skewness etc.. were extracted from each signals of different fault and analysed.
  • As per the test, the Vibration test showed best results tan any other sensors and have the ability to detect the faults at early stage.
An Intelligent Fault Diagnosis for Aircraft Fuel Systems Using ML Approaches:




This method was implemented on a laboratory scale fuel test rig developed in the IVHM centre in Cranfield University. The project included the testing of the rig and gathering the signals for healthy condition and different fault conditions injected in the rig and process them in MATLAB & Python for extracting different features and developing an artificial intelligence for diagnosing the faults in the system using machine learning approach in MATLAB.

  • The data from the pump speed, pressure sensors and flow meter for different fault conditions and healthy conditions were collected, cleaned and sorted into training and test data.
  • Different kinds of features were extracted from each signals and feeded into Linear Regression Model (Beacause it detected with high accuracy). The final model was tested with the testing data and the model detected faults at high accuracy of over 82%.
  • The developed Fault Diagnosis System has the ability to find the fault, it's severity and also provides diagnosis to the fault.
